Unicode Emojis are the ones you see on your iOS, Android or Windows Device. These will appear natively for the device that is browsing them.

How are Emoji's viewed?

Each device has it's own style of Emoji's. On iOS, Android and Windows these are viewed differently based on the device. The code however, is standard for all of them and is basic HTML unicode.

How do I use an Emoji?

Simply copy and paste the Unicode for your desired emoji, or copy the emoji directly. These can also have different sizes just by adjusting the font size.
